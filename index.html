TailWindCSS

  • Преимущества:
  • - Упрощает разработку сайтов
  • - Оптимизирует жизнь веб разработчика
  • - Позволяет писать CSS код внутри HTML
  • - Пафосно звучит
  • Недостатки:
  • - Не может приготовить завтрак в постель
  • - Бесполезен для работы на заводе
  • - Незначительны

(Обожаю Tailwind, все недостатки не были выписаны чтобы кого либо обидеть, а лишь с целью посмеяться и занять себя чем либо)
